# Test Case Viewer

Расширение для VSCode, которое позволяет просматривать и редактировать тест-кейсы в формате **Markdown** (`.md`) в удобном читаемом виде.

## Возможности

- Панель Viewer в Activity Bar (webview) для просмотра/редактирования тест-кейса
- Дерево тест-кейсов (по папкам или по категориям)
- Фильтры по owner/status/testType/tags и др.
- Вложения (копирование файлов в `_attachment/` рядом с тест-кейсом)
- Поддержка темной темы VSCode

## Установка

### Установка VSIX (для пользователей)

См. подробную инструкцию: [`docs/USER_GUIDE.md`](docs/USER_GUIDE.md).

### Из исходников (для разработчиков)

1. Клонируйте репозиторий
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Скомпилируйте проект:
   ```bash
   npm run compile
   ```
4. Нажмите F5 в VSCode для запуска расширения в новом окне Extension Development Host
5. В новом окне откройте JSON файл с тест-кейсом

### Упаковка расширения

Для создания .vsix файла:

```bash
npm install -g @vscode/vsce
vsce package
```

Затем установите через:
```
code --install-extension test-case-viewer-0.0.1.vsix
```

## Использование

1. Откройте `*.md` файл с тест-кейсом в VSCode
2. В боковой панели (Activity Bar) нажмите на иконку **Test Case Viewer**
3. В панели Viewer отобразится содержимое открытого тест-кейса (и его можно редактировать)
4. При переключении между файлами/изменении файла содержимое панели обновляется автоматически

Подробное руководство пользователя: [`docs/USER_GUIDE.md`](docs/USER_GUIDE.md).
